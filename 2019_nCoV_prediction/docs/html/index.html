Introduction

2019_nCoV 每日感染人数预测程序

Dependencies

  1. 用到的C++相关库: G2O GLOG(含youibot log system) GFLAGS
  2. 用到的python功能包: math matplotlib

Note Items

  1. 计算机上需要预先安装G2O。
  2. 日志系统使用的是公司的日志系统，暂不开源，请谅解。如果要在自己的计算机上运行，可将代码中的日志替换为C++标准输出std::cout<<,并取消main中Glogger对象的定义即可。

Installation

Just as the follows

Step 1: install g2o and youibot log system from github

Step 2: install math and matplotlib packages for python

Step 3: install the application

$ cd 2019_nCoV_prediction
$ mkdir build
$ cd build
$ cmake ..
$ make
